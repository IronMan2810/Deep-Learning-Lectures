- Ակտիվացիոն ֆունկցիաներ և կորստի ֆունկցիաներ
- Վերահսկվող ուսուցում և նեյրոնային ցանցեր
- Տվյալների նախապատրաստում և նորմավորում
- Օպտիմիզացման ալգորիթմներ
- Փաթույթային նեյրոնային ցանցեր

- Ուշադրության մեխանիզմներ
